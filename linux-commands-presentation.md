## Slide 2 — Introduction to Linux
- Linux is an open-source operating system.
- It is widely used in servers, cloud, and development.
- Linux is powerful, secure, and flexible.

## Slide 3 — Why Linux Commands Matter
- Help users interact with the system
- Save time through fast operations
- Important for developers, DevOps, and system administrators
- Make automation easier

## Slide 4 — Basic Navigation Commands
- `pwd` — show current directory
- `ls` — list files and folders
- `cd` — change directory
- `clear` — clear terminal screen

## Slide 5 — File and Directory Commands
- `touch` — create a file
- `mkdir` — create a directory
- `cp` — copy files or folders
- `mv` — move or rename files
- `rm` — remove files or folders

## Slide 6 — Viewing File Content
- `cat` — display file content
- `less` — view file page by page
- `head` — show first lines of a file
- `tail` — show last lines of a file

## Slide 7 — File Editing Commands
- `nano` — simple text editor
- `vim` — advanced text editor
- `echo` — write text into a file

## Slide 8 — Permission Commands
- `chmod` — change file permissions
- `chown` — change file ownership
- Permissions help secure files and folders

## Slide 9 — System Information Commands
- `whoami` — current user
- `uname -a` — system information
- `hostname` — system name
- `date` — current date and time

## Slide 10 — Process Management Commands
- `ps aux` — show running processes
- `top` — monitor system performance
- `kill` — stop a process
- `pkill` — kill process by name

## Slide 11 — Networking Commands
- `ping` — check connectivity
- `ip a` — show network information
- `curl` — fetch data from a URL
- `wget` — download files

## Slide 12 — Advanced Linux Commands
- `grep` — search text
- `find` — locate files
- `awk` — process text
- `sed` — edit text from the command line
- `tar` — archive files

## Slide 13 — Real Life Uses of Linux Commands
- Software development
- Server management
- Cloud and DevOps
- Cybersecurity
- Automation scripts

## Slide 14 — Advantages of Linux Commands
- Fast and efficient
- Powerful control over the system
- Useful for automation
- Essential for technical careers

## Slide 15 — Conclusion
- Linux commands are important for beginners and professionals
- Learning commands improves productivity
- Practice is the key to mastering Linux
